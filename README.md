# UDR

Implements 3gpp 29.504 specification. Provides service
to PCF, UDM. UDR supports SBI interface and any other network function can use the 
service.

## UDR block diagram
![UDR Block Diagram](/docs/images/README-UDR.png)

## Upcoming changes
